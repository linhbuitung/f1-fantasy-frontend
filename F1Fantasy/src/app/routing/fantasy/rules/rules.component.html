WooF1 Fantasy Game Rules

1. Team Composition

  • Each player selects 5 drivers and 2 constructors.
  • Total team value must not exceed $100.
  • Driver and constructor prices may change over time.

2. Transfer Rules

  • Each race week allows 2 free transfers.
  • Additional transfers incur a 10-point deduction per extra transfer.
  • Transfers are locked 1 hour before qualifying.

Captain Bonus

  • One driver per race can be selected as Captain.
  • All points (positive or negative) for the captain are doubled.

4. Powerups

  • Only one powerup can be used per race.
  • Each powerup can be used once per season.

5. League System

  • Private Leagues: Created by users, invite-only, max 5 per user.
  • Public Leagues: Created by admins, open to all.
  • Leaderboards show total points and race-specific rankings.

6. AI Predictions

  • Earn AI usage by logging in daily (1 use per 14 consecutive days).
  • Use AI to predict qualifying grid or race results.
  • AI uses historical data from jolpica-f1 API.

Note: All rules are subject to change by admins. Please check this page regularly for updates.
